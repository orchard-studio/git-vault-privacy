# プライバシーポリシー

**最終更新日: 2026年2月21日**

## 概要

GitVault（以下「本アプリ」）は Orchard Studio が開発しています。本プライバシーポリシーは、本アプリがお客様の情報をどのように扱うかについて説明します。

## データの収集

GitVault は個人データを収集、送信、または第三者と共有することは**一切ありません**。本アプリはお客様のデバイス上でのみ動作し、以下のローカルデータを保存します。

### ローカルデータの保存

- **リポジトリメタデータ**: リポジトリ名、URL、ブランチ情報、ファイルインデックスは、デバイス上のローカル SQLite データベースに保存されます。
- **ファイルコンテンツキャッシュ**: ダウンロードした Markdown ファイルの内容は、オフライン閲覧用にローカルにキャッシュされます。
- **GitHub アクセストークン**: プライベートリポジトリを設定した場合、GitHub パーソナルアクセストークンは iOS Keychain に安全に保存されます。
- **アプリ設定**: テーマ、フォントサイズ、デフォルトブランチなどの設定は UserDefaults（SharedPreferences）に保存されます。

### ネットワーク通信

本アプリは GitHub API（`api.github.com`）とのみ通信を行い、以下の目的で使用します：
- リポジトリのファイルツリーの取得
- ファイルコンテンツのダウンロード
- リポジトリアクセスの検証

すべてのネットワークリクエストは、お客様のデバイスから GitHub のサーバーに直接送信されます。中間サーバーを経由したり、データが保存されることはありません。

## トラッキングなし

- 本アプリは分析やトラッキングサービスを**使用しません**。
- 本アプリに**広告は含まれません**。
- 本アプリは Cookie やトラッキングピクセルを**使用しません**。
- 本アプリはデバイス識別子を**収集しません**。

## アプリ内課金

本アプリは買い切り型の Pro アップグレードを提供しています。購入取引は Apple の App Store によって完全に処理されます。本アプリは購入状態を示すローカルフラグのみを保存します。

## データの削除

本アプリが保存するすべてのデータは以下の方法で削除できます：
- 設定画面の「キャッシュをクリア」機能を使用して、ダウンロードしたリポジトリデータを削除
- デバイスからアプリを削除すると、Keychain のエントリを含むすべてのローカルデータが削除されます

## お子様のプライバシー

本アプリは13歳未満のお子様から意図的に情報を収集することはありません。

## ポリシーの変更

本プライバシーポリシーは随時更新される場合があります。変更は更新日とともにこのページに掲載されます。
